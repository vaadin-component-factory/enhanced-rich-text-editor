## Special Considerations

**Cell Metadata Format:**
Must be clearly documented:
```
{tableId}|{rowId}|{cellId}|{rowspan}|{colspan}|{templateId}|{uniqueId}
```
- What each field means
- When/how to use it
- Programmatic access
- Custom templates

**Delta Format:**
Table representation in Delta must be explained:
```json
{
  "insert": {
    "td": "table-1|row-1|cell-1|1|1||"
  }
}
```

**Integration Points:**
- How tables work with readonly sections
- How tables work with placeholders
- Sanitizer interaction
