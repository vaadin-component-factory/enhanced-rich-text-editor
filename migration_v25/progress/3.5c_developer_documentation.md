# Phase 3.5c: Developer Documentation

**Status:** NOT STARTED

## Objective

Create comprehensive developer documentation for extending ERTE 2, understanding its architecture, and contributing to the project.

## Scope

### Document Contents

1. **Architecture Overview**
   - ERTE shell structure (JS subclass + Java bridge)
   - RteExtensionBase pattern
   - Toolbar slot injection mechanism
   - Value format architecture (HTML-primary + Delta wrapper)
   - Sanitizer architecture (client + server)
   - Quill lifecycle integration

2. **Extending ERTE**
   - Creating custom blots
   - Adding custom formats
   - Extending the toolbar
   - Custom keyboard shortcuts
   - Using extendOptions hook
   - Theme customization

3. **Custom Blot Development**
   - Quill 2 / Parchment 3 API
   - Embed vs Inline vs Block
   - Guard node handling
   - Sanitizer integration
   - Testing custom blots

4. **Sanitizer Configuration**
   - Client-side sanitizer (ERTE_PRESERVED_CLASSES)
   - Server-side sanitizer (erteSanitize)
   - Whitelisting custom classes
   - Security considerations

5. **Keyboard Shortcuts**
   - Adding custom bindings
   - Overriding defaults
   - Keyboard module integration
   - shortKey handling (Ctrl/Cmd)

6. **Styling & Theming**
   - Lumo integration
   - CSS custom properties
   - Shadow DOM styling
   - Custom toolbar icons

7. **Building from Source**
   - Development setup
   - Build commands
   - Running tests
   - Debugging

8. **Contributing**
   - Code style
   - Testing requirements
   - PR process
   - Architecture decisions
