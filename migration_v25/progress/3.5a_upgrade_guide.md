# Phase 3.5a: Upgrade Guide ERTE 1→2

**Status:** NOT STARTED

## Objective

Create a comprehensive upgrade guide for users migrating from ERTE 1 (Vaadin 24 / Quill 1) to ERTE 2 (Vaadin 25 / Quill 2).

## Scope

### Document Contents

1. **Prerequisites & Dependencies**
   - Vaadin 24→25 upgrade requirements
   - Java version requirements (Java 21+)
   - Maven coordinate changes (ERTE 5.x → 6.x)
   - Spring Boot 3→4 considerations

2. **Breaking Changes**
   - Primary value format (Delta → HTML)
   - `asDelta()` wrapper API
   - Deprecated methods (extendQuill/extendEditor)
   - Removed features (if any)
   - API signature changes

3. **Feature Migration Guide**
   - Feature-by-feature code examples (V24 → V25)
   - Configuration changes
   - Event handling differences
   - Styling/theming updates

4. **Known Limitations**
   - Quill 2 embed guard node effects
   - History/undo behavior changes
   - Copy-paste limitations
   - Browser compatibility notes

5. **Migration Strategy**
   - Step-by-step migration process
   - Testing recommendations
   - Rollback considerations
   - Gradual vs. big-bang approach

6. **Troubleshooting**
   - Common migration issues
   - Debugging tips
   - Where to get help

## Success Criteria

- [ ] All breaking changes documented with code examples
- [ ] Each of 20 features has migration guidance
- [ ] Clear step-by-step migration path
- [ ] Known issues and workarounds listed
- [ ] Reviewed by docs-engineer agent
